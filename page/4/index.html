CSS3实战 笔记7-渐变设计

gradient

  • type 定义渐变类型，linear(线性渐变) 或 radial(径向渐变)
  • point 定义渐变的起点和结束点，支持 数值、百分比、关键字等，例如 0,0 left,top。
  • [radius] 当定义径向渐变时，设置渐变的长度，参数为数值。
  • [stop] 定义渐变色和步长，包含三个类型值，开始的颜色使用 from(color value)，结束的颜色使用 to(color value)，颜色的步长使用 color-stop(value, color value)。

CSS3实战 笔记5-完善的和模型和UI设计

border-color

  • CSS3 增加了这个属性，使用它可以为边框设置更多的颜色。
  • 如果只为该属性设置一个颜色值时，表示宽边设置为纯色，如果设置N个值，且边框宽度为N像素，那么可以在该边框上使用N中颜色，每个颜色显示1像素的宽度，如果边框宽度为10个像素，但是只声明了5种颜色，那么最后一种颜色用于剩下的宽度。
  • 可设置四个边框的颜色属性
    • border-top-color
    • border-right-color
    • border-bottom-color
    • border-left-color

2015年在阿里巴巴的面试经历

意外的开始

在年终前离开上家公司，只是觉得有愧于自己，虽然产品开发完成了，但离自己理想的结果相差太远，深刻的明白一个人的能力真的渺小。

离开后如愿找到一家小团队公司，想要实现未能实现的理想，正如预料的那样，新公司存在各种问题，没关系，我付出我的一份努力，会变好的。

入职几天后陆续收到杭州阿里、深圳阿里、广州UC的面试邀请，很意外，我并没有投，而且对自己的实力并不自信，我没有一个好的学历，甚至没有毕业证，所有能力都是自学的，面试官说并不重要，考虑后接受了杭州阿里的邀请。

接着跟BOSS坦白，希望辞职去面试，觉得偷偷去面试是很不尊重的，BOSS表示如果没有通过的话，就回来上班吧。

我不相信本命年的说法，也抗拒被观念束缚，但是今年真的很意外的好运气。（后来才知道本命年原来是不好的意思，Orz）

看不清的杭州

请了3天假，却没能好好复习，到杭州的前一天感觉脑袋还是空的，只是不断的在胡思乱想，寒冷的天气，灰蒙蒙的感觉不到色彩。

一开始遇到的面试官是清羽，很文艺很有亲和力的一个人，工牌显得很旧，我想对方应该是很资深的员工了。

CSS3实战 笔记4-新增的弹性盒模型

  • 引入了新的盒模型处理机制，该模型用于决定元素在盒子中的分布方式以及如何处理盒子的可用空间。

  • css3 新增了8个属性

    • box-align 定义子元素在盒子内垂直方向上的空间分配方式
    • box-direction 定义盒子的显示顺序
    • box-flex 定义子元素在盒子内的自适应尺寸
    • box-flex-group 定义子元素群组
    • box-lines 定义子元素分列显示
    • box-ordinal-group 定义子元素在盒子内的显示位置
    • box-orient 定义盒子分布的坐标轴
    • box-pack 定义子元素在盒子内水平方向的空间分配方式
